You are about to choose your Covr Code
Your Covr Code is used to verify and deny requests. It is the only code you have to remember for all your logins and transactions when using Covr.
You are the only one who knows it. Without it, no one can access your transactions in Covr.
BE SMART!
Choose a number sequence that you can easily remember but that others cannot easily guess.
DO NOT FORGET IT!
If you forget your Covr Code you can NOT recover it. You will have to delete the Covr app, re-install it and start over. All your connections will be secured.
DO NOT SHARE IT!
If by accident you reveal your Covr Code or otherwise suspect that someone else is using it, you should immediately change the code. You can do so in the Covr Settings.
